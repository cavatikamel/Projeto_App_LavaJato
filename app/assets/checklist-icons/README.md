# Icones simples do checklist

Conjunto vetorial simples para os estados do checklist LavaPrime.

Arquivos:

- `conforme.svg`: check verde.
- `arranhado.svg`: zigue-zague horizontal amarelo.
- `arranhado-azul.svg`: alternativa com duas linhas zigue-zague horizontais azuis.
- `amassado.svg`: linha ondulada verde.
- `quebrado.svg`: zigue-zague diagonal vermelho.
- `faltando.svg`: circulo com linha horizontal vermelha.
- `faltante.svg`: alias visual do estado faltando.
- `nao-aplicavel.svg`: circulo cinza com diagonal.
- `nao-verificado.svg`: interrogacao azul.
- `preview.svg`: visao geral do conjunto.

Padrao:

- Formato redondo.
- Fundo branco.
- Borda clara `#D7E5EB`.
- Traço grosso, simples e legivel em tamanho pequeno.
